## Notes

### Naming Conventions

- **Semantic**: `--semantic-{category}-{subcategory}-{property}`
- **Components**: `--component-{component}-{property}-{variant}`
- **Greenshift**: Uses WordPress standards `--wp--{preset|custom|style}--{path}`

### Mode Handling

- **Semantic**: Light/Dark modes use the same variable names (values change by mode)
- **Components**: Single Default mode
- **Greenshift**: Light/Dark modes use the same variable names (values change by mode)

### Usage in Figma

To add code syntax to variables in Figma:
1. Select the variable in the Variables panel
2. Click "Code syntax" field
3. Paste the corresponding `var(--variable-name)` from this table
4. Press Enter to save

### Pixel Units

For numeric values (spacing, radius, sizes), append `px` in your CSS:
```css
padding: var(--component-button-padding-x)px;
border-radius: var(--component-button-radius)px;
```
